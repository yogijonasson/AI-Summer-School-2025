Liste des variables avec leur signification

    class (classe - variable cible)
        e : edible (comestible)
        p : poisonous (toxique)

    cap-shape (forme du chapeau)
        b : bell (en cloche)
        c : conical (conique)
        x : convex (convexe)
        f : flat (plat)
        k : knobbed (en bouton)
        s : sunken (enfoncé)

    cap-surface (surface du chapeau)
        f : fibrous (fibreuse)
        g : grooves (striée)
        y : scaly (écailleuse)
        s : smooth (lisse)

    cap-color (couleur du chapeau)
        n : brown (marron)
        b : buff (jaune brunâtre)
        c : cinnamon (cannelle)
        g : gray (gris)
        r : green (vert)
        p : pink (rose)
        u : purple (violet)
        e : red (rouge)
        w : white (blanc)
        y : yellow (jaune)

    bruises (présence de meurtrissures)
        t : bruises (présent)
        f : no bruises (absent)

    odor (odeur)
        a : almond (amande)
        l : anise (anis)
        c : creosote (créosote)
        y : fishy (poisson)
        f : foul (mauvaise)
        m : musty (moisi)
        n : none (aucune)
        p : pungent (piquante)
        s : spicy (épicée)

    gill-attachment (attache des lamelles)
        a : attached (attachée)
        d : descending (descendante)
        f : free (libre)
        n : notched (dentée)

    gill-spacing (espacement des lamelles)
        c : close (proche)
        w : crowded (dense)
        d : distant (éloigné)

    gill-size (taille des lamelles)
        b : broad (large)
        n : narrow (étroite)

    gill-color (couleur des lamelles)
        k : black (noir)
        n : brown (marron)
        b : buff (jaune brunâtre)
        h : chocolate (chocolat)
        g : gray (gris)
        r : green (vert)
        o : orange (orange)
        p : pink (rose)
        u : purple (violet)
        e : red (rouge)
        w : white (blanc)
        y : yellow (jaune)

    stalk-shape (forme du pied)
        e : enlarging (élargissant)
        t : tapering (rétrécissant)

    stalk-root (type de racine du pied)
        b : bulbous (bulbeux)
        c : club (massue)
        u : cup (en coupe)
        e : equal (égal)
        z : rhizomorphs (rhizomorphe)
        r : rooted (enraciné)
        ? : missing (donnée manquante)

    stalk-surface-above-ring (surface du pied au-dessus de l'anneau)
        f : fibrous (fibreuse)
        y : scaly (écailleuse)
        k : silky (soyeuse)
        s : smooth (lisse)

    stalk-surface-below-ring (surface du pied en-dessous de l'anneau)
        f : fibrous (fibreuse)
        y : scaly (écailleuse)
        k : silky (soyeuse)
        s : smooth (lisse)

    stalk-color-above-ring (couleur du pied au-dessus de l'anneau)
        n : brown (marron)
        b : buff (jaune brunâtre)
        c : cinnamon (cannelle)
        g : gray (gris)
        o : orange (orange)
        p : pink (rose)
        e : red (rouge)
        w : white (blanc)
        y : yellow (jaune)

    stalk-color-below-ring (couleur du pied en-dessous de l'anneau)
        mêmes valeurs que stalk-color-above-ring

    veil-type (type de voile)
        p : partial (partiel)
        u : universal (universel)
        (Note : cette variable ne prend qu'une seule valeur 'p' dans le dataset, elle n'apporte donc pas d'information utile.)

    veil-color (couleur du voile)
        n : brown (marron)
        o : orange (orange)
        w : white (blanc)
        y : yellow (jaune)

    ring-number (nombre d'anneaux)
        n : none (aucun)
        o : one (un)
        t : two (deux)

    ring-type (type d'anneau)
        c : cobwebby (toile d'araignée)
        e : evanescent (évanescent)
        f : flaring (évasé)
        l : large (large)
        n : none (aucun)
        p : pendant (pendant)
        s : sheathing (gaine)
        z : zone (zone)

    spore-print-color (couleur de l'impression des spores)
        k : black (noir)
        n : brown (marron)
        b : buff (jaune brunâtre)
        h : chocolate (chocolat)
        r : green (vert)
        o : orange (orange)
        u : purple (violet)
        w : white (blanc)
        y : yellow (jaune)

    population (population)
        a : abundant (abondante)
        c : clustered (regroupée)
        n : numerous (nombreuse)
        s : scattered (dispersée)
        v : several (quelques-unes)
        y : solitary (solitaire)

    habitat (habitat)
        g : grasses (herbes)
        l : leaves (feuilles)
        m : meadows (prés)
        p : paths (chemins)
        u : urban (urbain)
        w : waste (déchets)
        d : woods (forêts)
